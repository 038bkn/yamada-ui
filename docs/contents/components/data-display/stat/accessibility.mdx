## ARIA Roles and Attributes

| Component  | Role and Attributes | Usage                                                                                                  |
| ---------- | ------------------- | ------------------------------------------------------------------------------------------------------ |
| `Stat`     | `icon`              | If set to `"increase"` or `"decrease"`, it is passed as `type` to `StatIcon`.                          |
| `StatIcon` | `type`              | Can be set to `"increase"` or `"decrease"`.                                                            |
|            | `aria-label`        | If `type` is set to `"increase"`, it will be `"Increased by"`. Otherwise, it will be `"Decreased by"`. |
