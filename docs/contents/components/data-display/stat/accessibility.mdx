## ARIA Roles and Attributes

| Component  | Role and Attributes | Usage                                                                                                               |
| ---------- | ------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `StatIcon` | `aria-label`        | Set to `"Increased by"` if `type` is set to `"increase"`, and to `"Decreased by"` if `type` is set to `"decrease"`. |
