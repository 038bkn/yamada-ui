## ARIA Roles and Attributes

| Component     | Role and Attributes | Usage                                                                                                                                                                                                                                                                                                                                                |
| ------------- | ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Avatar`      | `role="img"`        | Indicates the component is an image.                                                                                                                                                                                                                                                                                                                 |
|               | `name`              | The name of the person represented by the avatar.                                                                                                                                                                                                                                                                                                    |
|               | `alt`               | The alt text for the avatar image.                                                                                                                                                                                                                                                                                                                   |
|               | `src`               | The source URL of the avatar image.                                                                                                                                                                                                                                                                                                                  |
|               | `aria-label`        | Sets the name if `name` is provided and `src` is not set or while the avatar image have not completed to load. Sets the alt text if `alt` is provided and `src` is not set or while the avatar image have not completed to load. Defaults to `"Avatar Icon"` if neither `alt` nor `src` is set or while the avatar image have not completed to load. |
| `AvatarGroup` | `role="group"`      | Indicates the component is an group.                                                                                                                                                                                                                                                                                                                 |
