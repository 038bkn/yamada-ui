## ARIAロールと属性

| コンポーネント | ロールと属性               | 使い方                                                                                               |
| -------------- | -------------------------- | ---------------------------------------------------------------------------------------------------- |
| `AvatarName`   | `role="img"`               | `src`に設定された画像が読み込み未完了の場合、`src`が設定されてない場合に、画像であることを示します。 |
|                | `aria-label=name`          | `name`が設定されている場合に、名前が設定されます。                                                   |
| `AvatarImage`  | `role="img"`               | 画像であることを示します。                                                                           |
|                | `aria-label="Avatar Icon"` | `aria-label`が設定されていない場合に、`"Avatar Icon"`が設定されます。                                |
