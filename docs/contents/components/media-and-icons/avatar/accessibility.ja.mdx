## ARIAロールと属性

| コンポーネント | ロールと属性 | 使い方                                                                                                                                                                                                                                                                                                                                                                                    |
| -------------- | ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Avatar`       | `role="img"` | このコンポーネントが画像であることを示します。                                                                                                                                                                                                                                                                                                                                            |
|                | `name`       | アバターが表す人物の名前。                                                                                                                                                                                                                                                                                                                                                                |
|                | `alt`        | アバター画像の代替テキスト。                                                                                                                                                                                                                                                                                                                                                              |
|                | `src`        | アバター画像のソースURL。                                                                                                                                                                                                                                                                                                                                                                 |
|                | `aria-label` | `name` が設定されていて `src` が設定されていない場合、またはアバター画像の読み込みが完了していない間に名前を設定します。`alt` が設定されていて `src` が設定されていない場合、またはアバター画像の読み込みが完了していない間に代替テキストを設定します。`alt` も `src` も設定されていない場合、またはアバター画像の読み込みが完了していない間はデフォルトで `"Avatar Icon"` を設定します。 |
