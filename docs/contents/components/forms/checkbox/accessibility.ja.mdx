`Checkbox`は、アクセシビリティに関して[WAI-ARIA - Checkbox Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/checkbox)に従います。

`Fieldset`を使用してグループ化する場合は、以下の例が有効です。

```tsx live=false
<Fieldset>
  <Legend color="primary">好きなキャラクターを全て選んでください。</Legend>S
  <Checkbox value="孫悟空">孫悟空</Checkbox>
  <Checkbox value="ベジータ">ベジータ</Checkbox>
  <Checkbox value="フリーザ">フリーザ></Checkbox>
</Fieldset>
```

`FormControl`を使用しない場合は、`CheckboxGroup`に`aria-labeledby`を設定します。

```tsx live=false
<VStack gap="sm">
  <Heading as="h3" size="lg">
    キャラクター
  </Heading>

  <CheckboxGroup aria-labelledby="group">
    <Checkbox value="孫悟空">孫悟空</Checkbox>
    <Checkbox value="ベジータ">ベジータ</Checkbox>
    <Checkbox value="フリーザ">フリーザ</Checkbox>
  </CheckboxGroup>
</VStack>
```

## キーボード操作

| キー    | 説明                                                                                                                                                                     | 状態                     |
| ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------ |
| `Tab`   | 無効ではないチェックボックスにフォーカスします。                                                                                                                         | -                        |
| `Space` | フォーカスされたチェックボックスの状態をチェック済みと未チェックの間で切り替えます。                                                                                     | -                        |
|         | フォーカスされた`isIndeterminate={true}`が設定されているチェックボックスとそれによって制御されているチェックボックスの状態をチェック済み、未チェックの間で切り替えます。 | `isIndeterminate={true}` |

## ARIAロールと属性

| コンポーネント  | ロールと属性      | 使い方                                                                                                                                     |
| --------------- | ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| `CheckboxGroup` | `role="group"`    | チェックボックスのグループであることを示します。                                                                                           |
|                 | `aria-labelledby` | `CheckboxGroup`が`FormControl`内にあり、`FormControl`に`label`または`Label`が設定されている場合、その`id`を設定します。                    |
| `Checkbox`      | `id`              | `isIndeterminate={true}`が設定されたチェックボックスに関連付けるために使用する`id`。                                                       |
|                 | `aria-checked`    | チェックボックスがチェックされている場合は`"true"`を設定し、チェックされていない場合は`"false"`、中間状態の場合は`"mixed"`が設定されます。 |
|                 | `aria-controls`   | `isIndeterminate={true}`が設定されたチェックボックスによって制御されるチェックボックスの`id`を設定します。                                 |
