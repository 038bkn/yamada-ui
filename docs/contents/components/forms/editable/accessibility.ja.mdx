## ARIAロールと属性

| コンポーネント     | ロールと属性    | 使い方                                             |
| ------------------ | --------------- | -------------------------------------------------- |
| `EditableInput`    | `aria-invalid`  | Editableがinvalidの場合に`"true"`が設定されます。  |
|                    | `aria-disabled` | Editableがdisableの場合に`"true"`が設定されます。  |
|                    | `aria-readonly` | Editableがreadonlyの場合に`"true"`が設定されます。 |
|                    | `aria-required` | Editableがrequiredの場合に`"true"`が設定されます。 |
| `EditableTextarea` | `aria-invalid`  | Editableがinvalidの場合に`"true"`が設定されます。  |
|                    | `aria-disabled` | Editableがdisableの場合に`"true"`が設定されます。  |
|                    | `aria-readonly` | Editableがreadonlyの場合に`"true"`が設定されます。 |
|                    | `aria-required` | Editableがrequiredの場合に`"true"`が設定されます。 |
