`Alert` follows the [WAI-ARIA - Alert Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/alert/)for accessibility.

## Keyboard Navigation

No keyboard interaction needed.

## ARIA Roles and Attributes

| Component | Roles and Attributes    | Usage                                                                              |
| --------- | ----------------------- | ---------------------------------------------------------------------------------- |
| `Alert`   | `role="alert"`          | Indicates that it is an alert.                                                     |
|           | `aria-live="assertive"` | Indicates that it is an immediate notification.                                    |
|           | `aria-atomic=true`      | Indicates that it is the entire content of the alert element as the alert message. |
