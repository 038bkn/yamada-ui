`Alert`は、アクセシビリティに関して[WAI-ARIA - Alert Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/alert/)に従います。

## キーボード操作

キーボード操作は必要ありません。

## ARIAロールと属性

| コンポーネント | ロールと属性            | 使用方法                                                           |
| -------------- | ----------------------- | ------------------------------------------------------------------ |
| `Alert`        | `role="alert"`          | これはアラートであることを示します。                               |
|                | `aria-live="assertive"` | これは即時通知であることを示します。                               |
|                | `aria-atomic=true`      | アラート要素の全体の内容がアラートメッセージであることを示します。 |
