`Accordion`は、アクセシビリティに関して[WAI-ARIA - Accordion Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/accordion/)に従います。

## キーボード操作

| キー             | 説明                                                                                                         | 状態                                   |
| ---------------- | ------------------------------------------------------------------------------------------------------------ | -------------------------------------- |
| `Tab`            | フォーカスがアコーディオンに移動すると最初の項目に移動し、アコーディオン内である場合、次の項目に移動します。 | -                                      |
| `Shift` + `Tab`  | フォーカスがアコーディオン内である場合、前の項目に移動します。                                               | -                                      |
| `ArrowUp`        | 無効ではない前の項目にフォーカスします。最後の項目の場合、最初の項目にフォーカスをします。                   | -                                      |
| `ArrowDown`      | 無効ではない次の項目にフォーカスします。最初の項目の場合、最後の項目にフォーカスをします。                   | -                                      |
| `Space`, `Enter` | 無効ではない項目のパネルを展開します。                                                                       | -                                      |
|                  | 無効ではない項目のパネルを展開し、展開されている場合は折りたたみます。                                       | `isMultiple={true}`, `isToggle={true}` |
| `Home`           | 無効ではない最初の項目にフォーカスします。                                                                   | -                                      |
| `End`            | 無効ではない最後の項目にフォーカスします。                                                                   | -                                      |

## ARIAロールと属性

| コンポーネント   | ロールと属性      | 使い方                                                                                                                                        |
| ---------------- | ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `AccordionItem`  | `role="heading"`  | 見出しであることを示します。                                                                                                                  |
|                  | `aria-level`      | 1から6までの見出しのレベルを設定します。                                                                                                      |
|                  | `aria-disabled`   | パネルが展開されたときに`true`が設定されますが、ボタンによって折りたたむことはできません。他のパネルが展開されたときに`false`が設定されます。 |
| `AccordionLabel` | `type="button"`   | ボタンであることを示します。                                                                                                                  |
|                  | `aria-controls`   | 関連した`AccordionPanel`の`id`を設定します。                                                                                                  |
|                  | `aria-expanded`   | パネルが展開されたときに`true`が設定され、折りたたまれたときに`false`が設定されます。                                                         |
| `AccordionPanel` | `role="region"`   | ランドマーク領域を設定します。同時に展開される可能性のあるパネルが6個以上あるアコーディオンでは使用しないでください。                         |
|                  | `id`              | `AccordionLabel`と関連付けるために使用する`id`。                                                                                              |
|                  | `aria-labelledby` | 関連した`AccordionLabel`の`id`を設定します。                                                                                                  |
